About the Data

These two examples were created to test how quickly it would be for a designer and software engineer to make a visualisation based on data from very different sources. The visualisations are intended to be experiments of the API's power, rather than fully fledged visualisations. The data used were:

  • World Database on Protected Areas - Protected Areas Surface
  • Events or conflicts
  • Water basins in South Africa
  • Water Transitions
  • Population in South Africa
